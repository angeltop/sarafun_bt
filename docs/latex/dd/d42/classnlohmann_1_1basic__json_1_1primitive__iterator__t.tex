an iterator for primitive J\-S\-O\-N types 

This class models an iterator for primitive J\-S\-O\-N types (boolean, number, string). It's only purpose is to allow the iterator/const\-\_\-iterator classes to \char`\"{}iterate\char`\"{} over primitive values. Internally, the iterator is modeled by a {\ttfamily difference\-\_\-type} variable. Value begin\-\_\-value ({\ttfamily 0}) models the begin, end\-\_\-value ({\ttfamily 1}) models past the end.
